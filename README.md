## Краткое описание
Сборник различных утилитарных инструментов





## Возможности
- Поддержка **Delphi 12.1 Athens Community Edition** (предыдущие версии официально не поддерживаются)
- Официальная поддержка *Win32* и *Win64*, неофициальная поддержка остальных платформ доступных в текущей верии delphi

### Дополнительные инструменты для работы с массивами
- *Бинарный поиск* по неполному соответствию элемента
- *Линейный поиск* по неполному соответствию элемента
- *Расширенный динамический массив* - упрощённый лист для примитивов (Был взят из [QuickLib](https://github.com/exilon/QuickLib))





## Зависимости
### Необязательные [1]
- [QuickLib](https://github.com/exilon/QuickLib) (в тестах)

### Обязательные
- [DUnitX](https://github.com/VSoftTechnologies/DUnitX) (в тестах)

[1] Переключение возможно в [Common.inc](/includes/Common.inc) с помощью соответствующего ключа (не рекомендуется) или в настройках проекта путём добавления соответствующего ключа





## Использование
- Библиотека используется в виде отдельных модулей, подключение производиться в настройках проекта (указание путей поиска) или в настройках IDE (указание путей библиотек)
- Для управления настройками используется [Common.inc](/includes/Common.inc)
- В качестве примеров использования могут служить [тесты](/tests)
